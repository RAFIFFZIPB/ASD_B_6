Ketentuan :

Platform: Aplikasi berbasis Terminal/Console (CLI) dengan bahasa Python.
Penyimpanan Data: Wajib menggunakan File Handling (CSV/TXT) agar data tersimpan
permanen.

SPESIFIKASI TEKNIS (YANG HARUS ADA)
Setiap kelompok wajib membuat aplikasi dengan fitur minimal:

1. CRUD Lengkap: Aplikasi bisa Menambah, Membaca, Mengubah, dan Menghapus
   data.
2. Kombinasi Struktur Data: Minimal menggunakan dua jenis struktur data berbeda.
   ○ Contoh: Gunakan Linked List untuk data utama, dan Stack untuk fitur
   riwayat/history.
3. Fitur Pengolahan Data:
   ○ Sorting: Fitur untuk mengurutkan data (misal: urutkan nama A-Z atau harga
   termurah).
   ○ Searching: Fitur pencarian data spesifik.
4. Validasi Input: Program tidak boleh error/crash jika user salah ketik input.

Aplikasi Yang Dibuat :
Kelompok 6: Aplikasi Editor Teks Sederhana (Undo Feature)(TUMPUKAN & RIWAYAT (Fokus: Stack)
konsep LIFO (Last In First Out).)
○ Tantangan: Setiap mengetik kalimat, simpan ke Stack. Fitur "Undo" menghapus
kalimat terakhir.
